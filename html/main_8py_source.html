1# @file cotask.py
2# This file contains classes to run cooperatively scheduled tasks in a
3# multitasking system.
4#
5# Tasks are created as generators, functions which have infinite loops and call
6# @c yield at least once in the loop. References to all the tasks to be run
7# in the system are kept in a list maintained by class @c CoTaskList; the
8# system scheduler then runs the tasks' @c run() methods according to a
9# chosen scheduling algorithm such as round-robin or highest-priority-first.
