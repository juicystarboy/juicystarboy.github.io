Detailed Description

This file contains the main code to be run on Romi.

Tasks are defined as generators that are configured as finite state machines, then initialized as task objects using the cotask.Task class. Shares and queues are defined using the task_share.Share and task_share.Queue classes as needed to allow tasks to share information between each other.

This code causes Romi to follow a line and go around a box of prescribed size.

The general flow of the file goes as follows:
The line_read task uses the line sensors at the front of Romi to find the centroid of a line.
The line_follow task converts this centroid to a speed differential using a PID controller.
The motor_speed task converts the given speed to a duty cycle for each motor using a PID controller.
The motor_run task sets the duty of each motor

When the bump sensors detect an object, the go_around task is enabled and line_follow is disabled. The go_around task is a routine that consists of a series of motions that cause Romi to go around an object.
